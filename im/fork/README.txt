
Platform ：

	Unix/Linux platforms

Dependency :	

	GNU make tool

Compile :

	$ make

Run :
	Server NEEDS a port number which server is going to listen on

		$ ./server port

	User SHOULD pass server's ip address and port as arguments to client

		$ ./client server-ip port

NOTE
	The Chinese comments in the source file may appear messy if your platform not supported.
	You may get it right by using Virtual Studio 2008 or higer on Microsoft Windows Platforms.
